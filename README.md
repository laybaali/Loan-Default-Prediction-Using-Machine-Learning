Overview

Built a machine learning model to predict loan default risk using structured financial data, helping identify high-risk applicants.

Key Steps

Performed data cleaning, preprocessing, and EDA

Applied feature engineering and handled missing values

Addressed class imbalance using SMOTE

Built models: CatBoost, XGBoost

Evaluated using accuracy, precision, recall, and F1-score

Results

Ensemble models (CatBoost, XGBoost) performed best

Identified key factors influencing loan default

Improved model performance by handling class imbalance

Tech Stack

Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
